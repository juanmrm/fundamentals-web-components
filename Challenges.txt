- encapsulate component code
- reusable web components
- interoperable web components
- framework vs library
    
Un framework tiene generalmente:
    - gestion consistente del estado
    - router
    ...

Web component:
    - interoperable con framework Angular, Vue, React...

Interoperable:
    - indicar cuál debe de ser el estado del componente sólo mediante atributos y propiedades
    - puede escuchar eventos y responder a los eventos
    - puede emitir eventos para que otros componentes actuen
